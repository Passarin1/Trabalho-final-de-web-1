# Trabalho-final-de-web-1

## FitManager

Sistema web para gestão de uma academia, com páginas públicas, área do aluno e painel administrativo. O projeto foi desenvolvido em HTML, CSS e JavaScript puro, com navegação entre páginas e um fluxo de login/cadastro funcionando com armazenamento local no navegador.

## Estrutura criada

- Página inicial com apresentação da academia
- Páginas públicas: Sobre, Planos, Professores e Contato
- Página de login e cadastro
- Painel do aluno e painel administrativo
- Estilo responsivo e navegação comum em todas as páginas

## Como executar

1. Acesse a pasta do projeto.
2. Inicie um servidor local, por exemplo:
   - `python3 -m http.server 8000`
3. Abra o navegador em `http://127.0.0.1:8000/`
